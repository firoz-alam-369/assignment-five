What is event delegation in the DOM?

Event delegation is a technique in the DOM (Document Object Model) that allows you to handle events at a higher level in the DOM tree rather than attaching event listeners to individual elements. It takes advantage of event bubbling, where events propagate up from the target element to its ancestors.

What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM (Document Object Model) that allows events to propagate or "bubble up" from the target element to its ancestor elements. In other words, when an event is triggered on a specific element, it first occurs on that element, then it moves up the DOM tree and triggers the same event on each of its parent elements
